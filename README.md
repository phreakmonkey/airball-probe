# Airball Probe Firmware

### Hardware:
- STM32F103 Microcontroller
- Honeywell analog pressure sensors
- Bosch BMP085 or BMP180 atmospheric pressure sensor

### Dependencies
- Arduino 1.6+
- Libraries for STM32F series uC
- Adafruit BMP085 library
